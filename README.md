# Cheat-Sheet-git
Git cheat sheet

#### Checking your settings

`/home/user/.gitconfig`

`git config --list`

#### Cloning an Existing Repository

`git clone [url]`

#### Checking the Status of Your Files

`git status`

#### Tracking New Files

`git add`

#### Ignoring Files

`.gitignore`

#### Committing Your Changes

`git commit -a -v -m "Text"`
`-a let skip git add`
`-v add diff to the change`
`-m add a message`

#### Removing Files

`git rm`
